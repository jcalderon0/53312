## 💻 Requisitos Previos

Antes de ejecutar el analizador, tu computadora debe tener instalado:

| Herramienta | Versión mínima | Comando de verificación |
|-------------|----------------|--------------------------|
| Node.js     | 16 o superior  | `node -v`                |
| npm         | 6 o superior   | `npm -v`                 |
| Git (opcional) | cualquier   | `git --version`          |
| Java JRE    | 8 o superior   | `java -version` (solo para regenerar la gramática) |

- **Node.js**: Descargar desde [https://nodejs.org/] (versión LTS recomendada)
- **Git**: [https://git-scm.com/] (solo si clonarás el repositorio)
